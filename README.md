## Exploring the Effect of Hotspots on Average August Temperatures in 2023

### Objective
Show the relationship between hotspot activity and average August temperatures in 2023, as well as explore options for displaying data for visual deficiencies.

### Skills used
JavaScript  
Data visualization  
Raster Analysis 
Spatial data interpretation  
Observable mapping  
Iterative troubleshooting  

### Project Description
This project was a solo JavaScript and Observable visualization project focused on exploring how hotspot patterns relate to average August temperatures in 2023. My goal was to create a basic raster map showing temperature effects, alongside supporting non-geographic visualisations. Another aspect of this project was selecting and showing color schemes for users who suffer from different types of visual impairment.  This project helped me continue building my scripting and visualization skills outside of Esri platforms, while also improving my ability to work through troubleshooting and multiple rounds of revision.

# Group Projects
## ArcGIS Dashboard Project

### Objective
Visualize spatial data and patterns using an interactive ArcGIS dashboard.

### Skills used
ArcGIS Online  
Arc Pro 
Dashboard design  
Data visualization  
Spatial analysis
Data Sourcing 
Live data integration

### Individual Tasks
Vector layers
-Bike Path
-Parks 
-Wildland refuge
Dashboard creation 
Live weather tool

### Project Description
This project uses an ArcGIS Dashboard to present urban camping availability in Missoula, Montana. This was a group project; I undertook the majority of the dashboard work and some of the vector analysis. This project forced my group to justify our classifications. The urban camping ordinance for Missoula has some inherent room for interpretation. Our group leaned towards a conservative estimate of usable area. This was the first project that I had to grapple with transferring files from hard drives to the cloud to other users. It also helped me prepare for group workflow, a skill which I have been able to develop further since. Additionally, during the dashboard design, I added tools that users of the dashboard would find value in, with live temperature and weather indicators.
