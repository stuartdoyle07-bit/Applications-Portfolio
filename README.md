
## GIS Portfolio

This repository is a collection of my GIS work. I am a Forestry student with minimal background in scriptwriting or Geospatial analysis, so this repository shows a wide range of my learning.

The goal of this portfolio is to highlight my experience with spatial analysis and mapping using tools like ArcGIS Pro, Python, and Observable Mapping. 

# Projects 

## Nuclear Reactor Range Project

### Objective
Show the distances between nuclear waste storage and nuclear reactors with interactive elements.

### Skills used
JavaScript  
Vector analysis  
Data visualization  
Data source location  
Multiple iteration editing  

### Project Description
This project was a solo JavaScript data visualization project. This was my first non-Esri geospatial analysis project. My goal was to create a user experience that allowed interaction while showing the variable distances between nuclear waste storage and nuclear reactors. Additionally, there are non-geographic data analyses to support my exploration of this topic. This project took multiple rounds of troubleshooting to reach the final product, which helped me advance my scripting skills.

## Exploring the Effect of Hotspots on Average August Temperatures in 2023

### Objective
Show the relationship between hotspot activity and average August temperatures in 2023, as well as explore options for displaying data for visual deficiencies.

### Skills used
JavaScript  
Data visualization  
Spatial data interpretation  
Observable mapping  
Iterative troubleshooting  

### Project Description
This project was a solo JavaScript and Observable visualization project focused on exploring how hotspot patterns relate to average August temperatures in 2023. My goal was to create a basic raster map showing temperature effects, alongside supporting non-geographic visualisations. Another aspect of this project was selecting and showing color schemes for users who suffer from different types of visual impairment.  This project helped me continue building my scripting and visualization skills outside of Esri platforms, while also improving my ability to work through troubleshooting and multiple rounds of revision.
